# **App Name**: LinguaLog

## Core Features:

- Multi-Language Landing Page: A clean, minimal landing page with links to the six language sections: Japanese, English, Spanish, French, Portuguese, and Korean.
- Language-Specific Content: Dedicated directories for each language, such as /src/pages/en/, /src/pages/ja/, /src/pages/es/, /src/pages/ko/ etc., to maintain organized content.
- Hello World Posts: A sample 'Hello World' blog post in each language to populate each section and test the setup.
- Language Navigation: A navigation menu that allows users to switch between the different language versions of the blog easily.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to evoke trust and readability.
- Background color: Light blue (#E8EAF6) for a calm reading environment.
- Accent color: Indigo (#5C6BC0) to draw the eye.
- Body and headline font: 'Inter', a sans-serif for clean readability across different languages.
- Minimalist layout with a focus on readability; ample white space and clear sectioning.
- Simple, neutral icons for navigation and language selection; focus on universal symbols.
- Subtle transitions between language versions for a smooth user experience.
